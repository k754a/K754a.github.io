Getting Started

This is my documentation for my editor. This is for developers, so if you don't really want to edit or learn about how I made this, that's okay!


This editor is created in C++ and some C. It is lightweight and you can edit HTML, CSS, text, and Python files!

**Note: This editor can run other files, but the files above have built-in IntelliCode**

The editor right now cannot be compiled without issues with font files, so for now there are no compiled binaries.

All the files you need are included in the GitHub repository.




How to Install - Automatic

Please click the link to download my Installer. And just let it run.




How to Install - Manual

First, download it from my Github and extract the zip file.


If you don't have it installed already, install Visual Studio and open it.
